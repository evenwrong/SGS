# 张妈计算器说明
### [点击进入张妈计算器](https://evenwrong.github.io/SGS/张妈计算器.html "张妈计算器")
> + 点击按钮或键盘来输入点数，计算器会自动计算
> + 也可以按A、0、J、Q、K来输入1、10、J（10）、Q（12）、K（13）
## 更新日志

> + 若超过一分钟没有按键，下次按键时会自动清除之前的所有数字，以避免上次计算后忘记清除的情况
